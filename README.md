# OtoParça Bul

Sanayideki oto yedek parça satan küçük dükkanları ve parça arayan müşterileri buluşturan React + Vite + TypeScript prototipi.

## Kurulum

```bash
npm install
npm run dev
```

Ardından tarayıcıda terminalde görünen yerel adresi açın. Genellikle:

```bash
http://localhost:5173
```

## Komutlar

```bash
npm run dev
npm run build
npm run preview
```

## Notlar

- Backend, gerçek veritabanı, ödeme sistemi veya gerçek giriş sistemi yoktur.
- Tüm veriler `src/data/mockData.ts` içinde mock data olarak tutulur.
- WhatsApp butonları demo link formatındadır.
- Stok ekleme ve filtreleme gibi işlemler tarayıcı içi local state ile çalışır.
